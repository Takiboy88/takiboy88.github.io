Frequently Asked Questions

What is Relinx and what does it do?
Relinx is a powerful Android app designed to help you organize, categorize, and manage your important links and bookmarks. It provides smart search capabilities, custom categories, and tagging systems to keep your digital resources organized and easily accessible.
Is my data stored securely?
Yes, absolutely! Relinx stores all your data locally on your device using a secure SQLite database through Android's Room persistence library. Your data never leaves your device unless you explicitly choose to export it.
How do I create categories and tags?
You can create categories and tags when adding or editing a link:
  • Tap the "+" button to add a new link
  • Fill in the link details (title, URL, description)
  • Select or create a category from the dropdown
  • Add multiple tags separated by commas
  • Save your link with the new organization
Can I export my links to use in other applications?
Yes! Relinx supports exporting your entire link collection to JSON format. You can then import this data back into Relinx or use it with other applications that support JSON imports.
How does the search function work?
Relinx provides comprehensive search capabilities:
  • Search by link title or description
  • Search by URL or domain
  • Filter by specific categories
  • Search by tags
  • Combine multiple search criteria
Does Relinx require internet access?
Relinx requires internet access only when you want to open links in your browser or when ads are being displayed. All core functionality (adding, editing, searching, organizing links) works completely offline.
Is Relinx free to use?
Yes, Relinx is completely free to download and use. The app is supported by non-intrusive banner advertisements that help keep the app free while supporting ongoing development.
Can I use Relinx without seeing ads?
The current version includes banner ads to support development costs. The ads are designed to be non-intrusive and don't interfere with the app's functionality. Future updates may include ad-free options.

Troubleshooting Guide

Links won't open in browser
Solution: Check URL format and browser settings
  1. Ensure the URL includes "http://" or "https://" prefix
  2. Check that you have a default browser app installed
  3. Verify the URL is correctly formatted
  4. Try opening a different link to isolate the issue
  5. Restart the app and try again
App crashes when starting
Solution: Clear app data and restart
  1. Go to Android Settings > Apps > Relinx
  2. Tap "Storage" then "Clear Data"
  3. Restart your device
  4. Open Relinx again
  5. If the issue persists, reinstall the app
Search not returning expected results
Solution: Check search criteria and database
  1. Clear the search field and try a simpler query
  2. Check if the link exists in the expected category
  3. Try searching by different criteria (title, URL, tags)
  4. Ensure there are no typos in your search term
  5. Restart the app to refresh the database connection
Export/Import not working
Solution: Check file permissions and storage
  1. Ensure you have sufficient storage space
  2. Check that the app has file access permissions
  3. Try exporting to a different location
  4. Verify the JSON file format before importing
  5. Restart the app and try the operation again
Ads loading slowly or not displaying
Solution: Check network connection
  1. Verify you have a stable internet connection
  2. Try switching between WiFi and mobile data
  3. Clear the app cache in Android settings
  4. Restart the app to refresh ad connections
  5. Note that ad display doesn't affect core functionality
